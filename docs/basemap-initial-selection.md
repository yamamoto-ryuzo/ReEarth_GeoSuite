# ベースマップ初期選択（仕様・確認手順）

## 仕様
- プラグインは Inspector の `base:` 行を上から順に解析します。
- 初期表示されるベースマップは、Inspector に並んだ `base:` エントリの「上から最初のもの」を優先して選択するようになっています。
- ただし、パーマリンクやユーザ操作で `_lastAddedBasemapUrl` が設定されている場合はそちらが優先されます。

## 確認手順
1. Re:Earth Visualizer 上で本プラグインを読み込む／ウィジェットを開く。 
2. Inspector（一括設定）の `inspectorText` に複数の `base:` 行を上から順に記述する（例: OpenStreetMap を先頭）。
3. ウィジェットをリロードまたは `Refresh` を実行して設定を反映させる。
4. UI 上部のベースマップドロップダウンを確認する。初期で選択されているのが Inspector 上で上から最初に書いた `base:` のはずです。

## 備考
- もし "最後のもの" を初期にしたい場合、`processInspectorText` 実行後に `_lastAddedBasemapUrl` をその最後の `base:` に設定する変更を行う必要があります（今回のパッチは ``上から最初`` を意図しています）。
