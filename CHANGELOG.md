# Changelog

## v5.0.0
- インスペクタ一括設定を導入（`inspectorText`）：行ごとに `xyz:` / `info:` で設定可能に。
- 背景色をインスペクターで設定可能に。

## v4.0.0
- Info表示を追加：プラグインの Info タブで外部 HTML を表示できるように。

## v3.0.0
- Terrain / Shadow トグルを実装：地形表示と影表示の切り替え。

## v2.0.0
- レイヤー一覧 UI を追加：読み込んだレイヤーの表示/非表示や管理を容易に。

## v1.0.0
- XYZ タイル管理の基本機能を実装：XYZ タイルの追加・表示をサポート。
